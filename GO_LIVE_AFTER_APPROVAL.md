# GO LIVE AFTER APPROVAL — ai.pccc.vn

## Những việc chỉ làm sau khi owner duyệt
1. Chọn kiến trúc backend thật cho chat / auth / docs / tools
2. Chọn provider AI và chiến lược fallback
3. Thiết kế ingestion cho tài liệu PCCC / SOP / dữ liệu nội bộ
4. Thiết kế auth + workspace isolation thật
5. Thiết kế audit trail / role permissions
6. Lên kế hoạch domain / hosting / deployment
7. Chọn phạm vi MVP thương mại hóa đầu tiên
