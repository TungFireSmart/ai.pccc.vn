# UI REVIEW NOTES — ai.pccc.vn

- Có câu chuyện sản phẩm tương đối rõ
- Có nhiều màn hình thay vì một landing rời rạc

- Library / Enterprise còn thiên về giới thiệu, chưa đủ “wow”
