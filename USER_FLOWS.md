# USER FLOWS — ai.pccc.vn

## Flow 1 — Ask a compliance / regulation question
1. User opens Workspace
2. Selects `Pháp lý PCCC`
3. Pastes question or uploads relevant file
4. Receives answer + likely basis for follow-up work
5. Saves conversation into project workspace

## Flow 2 — Read a bid document
1. User opens Workspace
2. Selects `Hồ sơ thầu`
3. Uploads HSMT / BOQ / technical file
4. Asks for summary/checklist/risk extraction
5. Receives structured output for team action

## Flow 3 — Support sales response
1. User selects `Kinh doanh PCCC`
2. Provides customer question / project context
3. Requests response draft or consultation framework
4. Receives draft response + suggested next materials

## Flow 4 — Build internal knowledge capability
1. Admin/team uploads SOP / docs into Library
2. Content becomes part of internal knowledge base
3. Team uses Workspace and agents on top of that knowledge
