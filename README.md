# ai.pccc.vn

Mục tiêu: xây dựng website/app AI hàng đầu cho người làm việc trong lĩnh vực PCCC tại Việt Nam.

Trạng thái: pre-approval candidate đã freeze-for-review, sẵn để owner duyệt ngay.

## Nguyên tắc
- UI quen thuộc như ChatGPT, nhưng không sao chép thương hiệu.
- Tập trung vào nghiệp vụ PCCC Việt Nam.
- Ưu tiên chất lượng sản phẩm và kiến trúc mở rộng.
- Chưa phụ thuộc domain/hosting/dịch vụ ngoài ở giai đoạn duyệt nội bộ.

## Deliverables giai đoạn 1
1. Product blueprint
2. Information architecture + UX flows
3. Visual system + app shell
4. Frontend app prototype có thể chạy local
5. Bản duyệt nội bộ hoàn chỉnh

## Current status
- Product foundation docs: done
- Frontend prototype: running/building successfully
- Internal review build: complete and ready for owner review
- Latest UX pass: deeper chat shell, multi-state auth, and account/workspace/admin concepts added to the prototype
- Third-party go-live services: intentionally deferred until after approval
