# PHASE 2 KICKOFF — ai.pccc.vn

## Mục tiêu pha 2
Chuyển `ai.pccc.vn` từ bản duyệt nội bộ sang **MVP planning + technical foundation**.

Pha này không còn ưu tiên polish giao diện.
Ưu tiên là chốt:
- MVP v1 làm gì
- hệ thống thật gồm những khối nào
- data model v1 ra sao
- thứ tự build để làm nhanh nhưng không phải đập đi làm lại

## Kết quả cần đạt ở cuối pha 2
1. Có tài liệu scope MVP v1 rõ ràng
2. Có module map cho hệ thống thật
3. Có data model / DB foundation v1

5. Có build order rõ cho backend + frontend integration
6. Có thể bắt đầu implementation thật mà không còn mơ hồ phạm vi
